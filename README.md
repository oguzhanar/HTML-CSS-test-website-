This project is a web page study developed to practice and reinforce my theoretical knowledge while completing my HTML and CSS training. Focused on the BMW M series, it features both informative content and an interactive car configurator interface.

Project Features:
M4 Specifications: A semantic introductory section containing performance data for the BMW M4 model.

M Series Cards: A dynamic gallery layout styled with CSS Grid, listing all M models with their unique background images.

Design Your Own Monster: A form section and a horsepower table aligned side-by-side using CSS Flexbox layout.

Custom CSS Enhancements: Features fully custom-styled radio buttons, modern input fields, custom web font integrations, and informational utility cards with :hover effects.

Technologies Used:
HTML (Semantic tags, forms, and table elements)

CSS (Flexbox, Grid, Custom Fonts, Transition Effects)
